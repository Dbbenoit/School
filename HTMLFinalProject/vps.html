Virtual Private Servers (VPS)

What is a Virtual Private Server?

A Virtual Private Server Server, or VPS, is when a server hosts different clients by breaking the servers resources up into multiple virtual machinges. Each virtual machine has its own allocated CPU, RAM and storage. This is similar to shared hosting, but clients do not share resources and thus a clients service will not be impacted by other clients on the server. The resource allocation is done by a hypervisor. A hypervisor manages the virtual machines to make multiple virtual operating systems. The physical machine is called the host machine, and all of the virtual machines are called the guests.

It is common for a website to begin using shared hosting and move on to VPS. When a website is small and doesn't have much traffic, the shared resources of shared hosting can be enough, but as the popularity and expectations of a brand increase, having the improved perfomance and reliablity of dedicated resources can be desirable. Moving to VPS also allows the site holder to have better control over the scalability of the site. Some shared servers may limit the kind of softwared installed, like certaing plug ins. If a site manager finds that their site is running slowly or crashing, it may be time to move up to a VPS service.
