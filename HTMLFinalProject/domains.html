Domain Names

Domain Names are one of the most recognizable features of the internet. It is how we identify, label and communicate about a website. Domain Names are what we would coloqually call a website's name. From a technical standpoint, in human terms, it is the websites address. Computers locate each other using an IP address, or Internet Protocol. The IP address is large series of numbers and periods. This would be too hard for a person to remember, and would be almost imposible to market. To make things easier, a Domain Name is registered and understood set of alphanumeric characters associated with the IP address. This, idealy, would be a name or phrase that would be easy for aperson to remember, but doesn't have to be. Examples of well-known Domain Names would be Google.com, CCRI.edu, or RI.gov.
Notice the different suffixes following each of the identifying brands or organizations for the listed Domain Name. These are Top-Level Domain Names. These can help associate your Domain Name with its intended purpose. .com for commercial, .gov for governament and .edu for education.
The first part of each example is the second-level domain. This is the most specific part of your domain, as it is not shared with different organizations and is unique to a site.
A third level domain is used to separate different pages on a site. If the second-level domain is a street address, the third-level is the room number. These would usually be behind a slash after the first level domain.
Put them all together, and you have a website's URL!
Domain Name registrars are typically where an organization purchases a Domain Name from. There are many out there, including some big name brands, like GoDaddy, WIX, and Squarespace. Domain Name can be rented on a monthly basis, or purchased outright. A monthly plan can be as little as 1.99/month, but typically go for about 20/month. Purchasing a Domain Name can be in the thousands of dollars.
